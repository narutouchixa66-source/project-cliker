# Project - Telegram Mini App Кликер

## Описание
Telegram мини-приложение "Project" - кликер игра с системой заработка криптовалют.

## Основной функционал

### 🎮 Игровая механика
- **Кликер**: Заработок P монет кликами по большой желтой кнопке
- **Баланс**: Отображение текущего баланса P монет со значком ₽

### 💰 Система валют
- **P монеты**: Основная валюта, зарабатывается кликами
- **J монеты**: Премиум валюта, получается через обмен и сундуки
- **TON коины**: Конечная цель для вывода (в разработке)

**Курс обмена**: 10,000 P = 100 J

### 📦 Система сундуков
5 уровней редкости с разными наградами:

| Редкость | Реклама | P монеты | J монеты | Эмодзи |
|----------|---------|----------|----------|---------|
| Редкие | 3 | 100-200 | 10-100 | 📦 |
| Эпические | 6 | 200-300 | 20-150 | 🎁 |
| Мифические | 15 | 500-1000 | 40-700 | 💎 |
| Легендарные | 20 | 1000-2000 | 100-1000 | 👑 |
| Ультра | 30 | 0 | 300-5000 | 🌟 |

### 👥 Реферальная система
- **Бонус за реферала**: 500 J + 300 P монет
- **Уникальная ссылка**: Генерируется для каждого игрока
- **Статистика**: Отслеживание приглашенных и заработка

### 🎬 Рекламная система
- Просмотр рекламы для получения сундуков
- Ежедневный лимит: 50 просмотров
- Счетчик рекламы: 3/6/15/20/30 для разных сундуков

## Установка и запуск

### 1. Настройка Firebase
1. Создайте новый проект в [Firebase Console](https://console.firebase.google.com/)
2. Включите Realtime Database
3. Скопируйте конфигурацию в `script.js`
4. Настройте правила безопасности:

```json
{
  "rules": {
    "users": {
      "$uid": {
        ".read": "$uid === auth.uid || true",
        ".write": "$uid === auth.uid || true"
      }
    }
  }
}
```

### 2. Создание Telegram бота
1. Создайте бота через [@BotFather](https://t.me/botfather)
2. Получите токен бота
3. Настройте Web App для бота

### 3. Размещение
1. Загрузите файлы на хостинг (GitHub Pages, Netlify, Vercel)
2. Укажите URL в настройках бота
3. Тестируйте в Telegram
